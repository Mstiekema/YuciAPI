YuciAPI

Custom Twitch API for everyone

What is YuciAPI?

YuciAPI is a custom API created by Mstiekema. The API is here for every developer, bot hoster or Twitch user. With it you can easily retrieve info from Twitch users without using any authentication or password.

Endpoints

/followage/:user/:channel
Retrieves the followage of a certain user

/followsince/:user/:channel
Retrieves when a user followed a channel

/user/age/:user
Shows when a user's account was created

/user/id/:user
Shows the ID of a user

More coming soon

Examples

Here are a few examples of how you can add these API's to your bot. Replace the CAPITALIZED words with your own channel.

Nightbot
$(urlfetch http://api.yucibot.nl/followage/$(touser)/CHANNEL)

Ankhbot R2
$readapi(http://api.yucibot.nl/followage/$user/CHANNEL)

Deepbot
@customapi@[http://api.yucibot.nl/followage/@user@/CHANNEL]
