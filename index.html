YuciAPI

A custom Twitch API

What is YuciAPI?

YuciAPI is a custom API created by Mstiekema. The API is here for every developer, bot hoster or Twitch user. With it you can easily retrieve info from Twitch users without using any authentication or password.

Endpoints

URL What does it do?
/followage/:user/:channel Retrieves the followage of a certain user
/followsince/:user/:channel Retrieves when a user followed a channel
/user/age/:user Shows when a user's account was created
/user/id/:user Shows the ID of a user
/user/pf/:user Shows the URL of a user's profile picture
/user/followers/:user Returns the amount of followers of a channel
/user/game/:user Returns the current game
/user/title/:user Returns the title of the stream
/user/uptime/:user Returns the uptime of the stream
/user/viewers/:user Returns the live viewer count of the stream

Examples

Here are a few examples of how you can add these API's to your bot. Replace the CAPITALIZED words with your own channel.

Bot Example
Nightbot $(urlfetch http://api.yucibot.com/followage/$(touser)/CHANNEL)
Ankhbot $readapi(http://api.yucibot.com/followage/$user/CHANNEL)
Deepbot @customapi@[http://api.yucibot.com/followage/@user@/CHANNEL]
